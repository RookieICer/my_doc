Detailed Description

TLM 2.0 Generic Payload extension class used to model AXI transactions using a LT coding style. It corresponds to the ARM amba_pv_extension class. See the ARM AMBA-PV Extensions to OSCI TLM 2.0 Reference Manual for detailed documentation.

It is not designed to be used directly (for example, it is not randomizable). Rather, it is used to interface an AMBA-PV component to the corresponding SVT VIP.

Does not support user-defined attributes.
