Detailed Description

This class is used to model a single FIFO.

An instance of this class represents a FIFO in a single address location.

Internally, the memory is modeled with a queue of elements.

  function bit
 svt_axi_fifo_mem::read
 (  int index = -1, output logic [SVT_AXI_MAX_DATA_WIDTH-1:0] data  ) 

Returns the value of the data word stored by this object.

index - The index within the FIFO to be read. If -1 is provided, the element in front of queue is read and popped out of the queue. If index is not -1, the element at the given index is read, but is not popped out of the queue

data - The data stored at the given index. If the index does not exist or if the queue is empty, the data is not valid.

Return values - Returns 1 if the element at given index exists. If the index does not exist or if the queue is empty, returns 0.
