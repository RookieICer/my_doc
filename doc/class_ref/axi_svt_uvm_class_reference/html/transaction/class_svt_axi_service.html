Detailed Description

This class is a service transaction class:

  • used for monitoring low power interface. It captures the information on low power signals during the low power entry and low power exit handshakes
  • used as service transaction class to initiate sysco related service request on the axi service sequencer.

 svt_axi_service::axi_qos_accept_valid_ranges

Constraint on the qos servise request attributes.
constraint axi_qos_accept_valid_ranges {
    if (!cfg.qos_accept_signaling_enable)
    {
      !(service_type inside {QOS_ACCEPT_WRITE,QOS_ACCEPT_READ});
      write_qos_accept_level == 0;
