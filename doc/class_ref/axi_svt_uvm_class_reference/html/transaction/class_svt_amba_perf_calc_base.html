Detailed Description

Class used internally by the VIP to calculate performance. Typically an instance of this class is created by the VIP for each performance metric. At the end of each interval the values of this class are used for checking/reporting as well as updating an instance of svt_amba_perf_rec_base class which stores the performance summary for an interval

 real  attribute
 svt_amba_perf_calc_base::active_duration = 0

The active time on the bus i.e., when there is some transaction going on the bus. This is used for throughput calculation in AHB when perf_exclude_inactive_periods_for_throughput is enabled and perf_inactivity_algorithm_type is EXCLUDE_ALL. Depending on the kind of performance metric this will be updated.

 real  attribute
 svt_amba_perf_calc_base::duration = 0

The duration elapsed in a given interval. Depending on the kind of performance metric this may be updated with the max/min/aggregation of actual time elpased in the interval. For example, if the performance metric is throughput, this will be updated with the total time in the interval. If it is a performance metric measuring average latencies this will be updated with an aggregation of the latencies seen. If it is a performance metric on max/min duration, this will be updated with the curr max/min duration observed.
