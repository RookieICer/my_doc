Detailed Description

Port configuration class contains configuration information which is applicable to individual AXI low power master interface in the system. Some of the important information provided by port configuration class is:

  • Active/Passive mode of the low power component
  • Enable/disable protocol checks
  • Port configuration contains the virtual interface for the port
