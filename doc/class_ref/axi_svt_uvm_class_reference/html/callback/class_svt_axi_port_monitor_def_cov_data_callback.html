Detailed Description

The coverage data callback class defines default data and event information that are used to implement the coverage groups. The coverage data callback class is extended from callback class svt_axi_port_monitor_callback . This class itself does not contain any cover groups. The constructor of this class gets svt_axi_port_configuration handle as an argument, which is used for shaping the coverage.
