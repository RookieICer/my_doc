Detailed Description

State coverage is a signal level coverage. State coverage applies to signals that are a minimum of two bits wide. In most cases, the states (also commonly referred to as coverage bins) can be easily identified as all possible combinations of the signal. This Coverage Callback consists having covergroup definition and declaration. This class' constructor gets the port configuration class handle and creating covergroups based on respective signal_enable set from port configuartion class for optional protocol signals.
